# Constitutional Claude VS Code Setup

## 🚀 Quick Start
1. Install "Continue" extension in VS Code
2. Double-click `launch_vscode.bat`
3. Open any Python file
4. Press `Ctrl+I` for AI assistance

## 💡 Tips
- Select code → `Ctrl+Shift+I` to ask questions
- AI suggestions appear as you type
- Uses local Ollama models (no API keys)
